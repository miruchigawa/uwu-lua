# UwU Lua
Custom lua language with UwU Syntax

## Example
```
whiwe true do 
    print("Hewwo Wowwd :3")
end
```

Or see at [test/main.wua](test/main.wua)

Keywords on wua: and, break, do, if, ewse, ewseif, and, true, fawse, fow, fn, goto, if, in, local, nil, nyot, or, repeat, wetuwn, then, untiw, whiwe
